# Registration permissions

| Module       | Permission                     | Key                              | Description                                                                                                                                                                                                                                                                                                               | FE Permissions context              |
| ------------ | ------------------------------ | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| registration | View registration              | `registration-area-view`         | This is the ability to view the registration area on the menu item and access the registration, this is the overarching permission that you need to access the registration area.                                                                                                                                         | `registration.registration.canView` |
| registration | View Organisation              | `registration-view-organisation` | This is the ability to view organisations and teams within the registration area. If a league user has this permission, they can see the organisations and the squads within the association. If an organisation user has this permission, they will see the squads that their user has access to via the squad selector. | `registration.organisation.canView` |
| registration | View Athlete                   | `registration-view-athlete`      | This is the ability to view athletes within the registration area. If a league user has this permission, they will see the athletes within that association. If an organisation user has this permission, they will see the athletes within that organisation they have access to via the squad selector.                 | `registration.athlete.canView`      |
| registration | Edit Athlete                   | `registration-edit-athlete`      | The is the ability to upload files for athletes on behalf of the athletes. When the information is uploaded, it will state the user who uploaded it, and what organisation they are from.                                                                                                                                 | `registration.athlete.canEdit`      |
| registration | View Staff                     | `registration-view-staff`        | This is the ability to view staff within the registration area. If a league user has this permission, they will see the staff within that association. If an organisation user has this permission, they will see the staff within that organisation.                                                                     | `registration.staff.canView`        |
| registration | Edit Staff                     | `registration-edit-staff`        | The is the ability to upload files for staff on behalf of the staff. When the information is uploaded, it will state the user who uploaded it, and what organisation they are from.                                                                                                                                       | `registration.staff.canEdit`        |
| registration | View Registration Requirements | `registration-view-requirements` | This is a sub-permission that if you have the view athlete or view staff the user can see the requirements and the documents associated to that particular user type.                                                                                                                                                     | `registration.requirements.canView` |
| registration | Manage Registration Status     | `registration-manage-status`     | The ability to update the registration status, what statuses appear here is dependant on whether the user is on a league or organisation level.                                                                                                                                                                           | `registration.status.canEdit`       |
| registration | Manage Registration Payment    | `registration-manage-payment`    | This is the ability to pay for a set number of staff/athletes. This is behind a permission so leagues who do not require payment, will just not use this permission.                                                                                                                                                      | `registration.payment.canEdit`      |
| settings     | Manage Squads                  | `settings-manage-squads`         | The ability to view, create and edit squads                                                                                                                                                                                                                                                                               | TBD                                 |
| settings     | Manage Organisations           | `settings-manage-organisations`  | This is the ability to create and edit an organisation, this will only be able to be used for org types of League.                                                                                                                                                                                                        | TBD                                 |
| settings     | Manage Athletes                | TBD                              | This is an existing permission but will be used (and needs to be fleshed out) to control the MLS ID                                                                                                                                                                                                                       | TBD                                 |
| settings     | Manage Staff Users             | TBD                              | This is an existing permission but will be used (and needs to be fleshed out) to control the staff ID                                                                                                                                                                                                                     | TBD                                 |
