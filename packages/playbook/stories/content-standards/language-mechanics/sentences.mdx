# Sentences

Interface sentences instruct, confirm, or explain. Keep them direct.

## Imperative opening

Start with an action verb when guiding a task.

**Do** Add your first widget.
**Don't** You can add widgets now.

## Avoid permissive phrases

Remove "you can", "you may" unless clarifying permission.
