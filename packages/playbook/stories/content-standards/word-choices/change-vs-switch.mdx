# Change vs Switch

Use "Change" when replacing one item with another without altering broader context. Use "Switch" when moving between modes, contexts, or user accounts.

## Change

- Replacing one asset (e.g., headshot)

**Do** Change headshot
**Don't** Switch headshot (single asset)

## Switch

- Moving between squads, accounts, or modes

**Do** Switch squad
**Don't** Switch headshot (asset replacement)
