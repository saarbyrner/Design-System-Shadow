# Edit vs Manage

Use "Edit" for changing the properties of a single field or object. Use "Manage" for higher-level administration where multiple actions are possible.

## Edit

Place as inline link near the field.

**Do** Edit form
**Don't** Manage form (single field)

## Manage

Provide when entering a screen or modal with multiple settings or batch actions.

**Do** Manage collection channel
**Don't** Edit collection channel (multi-action area)
