# Select vs Choose

Use "Select" for constrained, explicit option sets where the choice is straightforward. Use "Choose" for subjective or open-ended decisions that require evaluation.

## Select

Scenarios:
- Dropdown menus
- Short lists of uniform items

**Do** Select session
**Don't** Choose session (dropdown)

## Choose

Scenarios:
- Larger inventories
- Strategic decisions

**Do** Choose collection method
**Don't** Select collection method (broad decision)
