# Create vs Add

Use "Create" when making something new from scratch (a dashboard, template). Use "Add" when inserting a new element into something existing (injury, graph).

## Create

**Do** Create dashboard
**Don't** Add dashboard (new artifact)

## Add

**Do** Add injury
**Don't** Create injury (existing system entity)
