# Onboarding

Help new users reach first value quickly.

## Checklist
- Highlight a single primary action.
- Use progressive disclosure for advanced steps.
- Reinforce benefits, not features.

Use `active voice` and concrete verbs (see `actionable-language`).

## Example
**Do** Add your team to start tracking wellness trends.
**Don't** Welcome to the athlete wellness dashboard platform.

**Do** Set a weekly wellness reminder.
**Don't** Configure wellness reminder settings now.
