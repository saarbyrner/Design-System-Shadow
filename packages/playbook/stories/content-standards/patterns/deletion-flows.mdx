# Deletion flows

Deletion removes data permanently or requires complex recovery. Keep copy minimal and clear.

## When to confirm

Confirm if:
- Action is irreversible
- Data loss affects other users
- Security or privacy implications exist

## Title pattern

Verb + object + question mark.

**Do** Delete session?
**Don't** Are you sure you want to delete this session?

## Body pattern

One sentence stating consequence.

**Do** This can't be undone.
**Don't** This action cannot be reversed. Are you sure you want to proceed?

## Actions

Primary: Delete. Secondary: Cancel.
