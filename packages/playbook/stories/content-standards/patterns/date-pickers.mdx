# Date pickers

Use "Done" to finalize a date selection when changes apply after confirmation. Use "Save" only if the date persists immediately outside a scoped picker.

## Button labels

**Do** Done
**Don't** Save (for inline picker modal)

## Instructions

Provide concise helper text only if needed: "Select start and end dates." Avoid directional language.
