## Playbook Icons

Icons in Playbook are a mixture of custom icons generated by Kitman Designers, and all the icons available in MUI.

We can use Icons in Playbook through the the `KitmanIcon` component and the `name` prop. See available names below

```jsx
import { KitmanIcon } from '@kitman/playbook/icons';

const IconUsage = () => <KitmanIcon name={KITMAN_ICON_NAMES.CheckCircle} />;
```

`KitmanIcon` accepts the same props as the `SVGIcon` in MUI, such as color and size. [Docs here](https://mui.com/material-ui/icons/#svgicon)

```jsx
<KitmanIcon name={KITMAN_ICON_NAMES.CheckCircle} />
<KitmanIcon name={KITMAN_ICON_NAMES.CheckCircle} color="primary"/>
<KitmanIcon name={KITMAN_ICON_NAMES.CheckCircle} color="secondary"/>
```
